# 重要点总结概览

## 1.jsx最终变为什么？

以前变为

```javascript
React.createElement()
```

最新的版本变为

```javascript
// Inserted by a compiler (don't import it yourself!)
import {jsx as _jsx} from 'react/jsx-runtime';
_jsx('h1', { children: 'Hello world' });
```
